\section*{Glossaire}

\begin{description}
  \item[API] \textit{Application Programming Interface} : interface permettant l'échange de données entre l'extension et le serveur.
  \item[Core Web Vitals] Ensemble d'indicateurs de performance orientés expérience utilisateur (ex. LCP, CLS).
  \item[Endpoint] Point d'accès HTTP (route) exposé par un service web, tel que \texttt{/api/traffic}.
  \item[Latence] Délai observé entre l'envoi d'une requête et la réception d'une réponse.
  \item[Polling] Mécanisme d'interrogation périodique d'un service pour récupérer des données actualisées.
  \item[Slowloris] Technique d'attaque applicative consistant à maintenir de nombreuses connexions incomplètes pour saturer un serveur.
  \item[Taux de succès] Proportion de requêtes aboutissant à un code HTTP de succès (2xx).
  \item[TTFB] \textit{Time To First Byte} : temps entre l'émission de la requête et la réception du premier octet de réponse.
\end{description}
